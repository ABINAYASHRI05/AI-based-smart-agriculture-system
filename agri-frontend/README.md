# 🌾 AI Smart Agriculture System — Frontend

React.js frontend for AI-powered Smart Agriculture System.

## Tech Stack
- React.js 18
- Chart.js + React-ChartJS-2
- Axios for API calls
- React Router DOM
- Web Speech API (Voice)

## Features
- Tamil + English bilingual UI
- Voice guide for farmers
- AI chatbot with voice + photo upload
- Satellite farm analysis dashboard
- Real-time market prices chart
- Government subsidies page
- Crop disease detection with photo upload

## Setup
git clone https://github.com/YOUR_USERNAME/smart-agriculture-frontend.git
cd smart-agriculture-frontend
npm install
npm start
# Opens on http://localhost:3000
